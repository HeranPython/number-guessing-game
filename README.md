# Number Guessing Game 🎯

A simple Python game where the computer randomly selects a number, and the player tries to guess it.

## How it works
- The computer picks a random number within a range.
- The player keeps guessing until they find the correct number.
- The program gives hints like "too high" or "too low".

## Features
- Random number generation
- User input handling
- Hint system (higher/lower)
- Loop until correct answer

## Skills used
- Python basics
- If/else statements
- While loops
- Random module

## Future improvements
- Add difficulty levels (easy, medium, hard)
- Add score system
- Limit number of tries
- Add GUI version with buttons
